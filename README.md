# Unity-SQLite
SQLite helper in Unity

Build In Unity 5.3.4f

## 用法 Usage

有两种方式可以使用，直接使用SQLiteHelper类或者继承并实现AbtractDataBase

There are two ways to use, direct use of SQLiteHelper class or inherited and implement AbtractDataBase
